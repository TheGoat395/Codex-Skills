# Demo Gallery

This repository includes a local proof-of-output gallery at `examples/premium-website-showcase/`.

These sites were produced as demonstrations of the Codex Premium Website Skills workflow: prompt/reference/spec in, polished local frontend artifact out.

## Local preview

Open this file after cloning the repo:

```text
examples/premium-website-showcase/gallery/index.html
```

## ZIP deliverables

The source demos are checked in as local preview folders. Generate zipped deliverables locally with:

```bash
python3 examples/premium-website-showcase/scripts/package_demos.py
```

Generated ZIPs are written to `examples/premium-website-showcase/demo-zips/`.
