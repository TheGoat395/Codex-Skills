# LinkFlow Hero Section

Modern SaaS-style hero section for a link/workflow automation product.

## Local preview

This demo is part of the Premium Website Showcase. The source artifact is tracked in the showcase manifest and can be packaged locally with:

```bash
python3 examples/premium-website-showcase/scripts/package_demos.py
```

## Why this demo exists

This demo shows how the website skill workflow can create polished SaaS-style hero sections quickly from brand/product direction.
