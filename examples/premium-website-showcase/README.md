# Premium Website Showcase

These demos are proof-of-output examples for the Codex Premium Website Skills library. They are not generic templates. They show the kind of polished local websites, hero sections, landing pages, and portfolios the skill collection is meant to help produce quickly from prompts, references, specs, and iterative QA.

## How to preview locally

Clone the repository, then open the gallery:

```text
examples/premium-website-showcase/gallery/index.html
```

Or open any demo directly:

```text
examples/premium-website-showcase/demos/michael-smith-portfolio/preview.html
```

No build step is required for the checked-in preview files.

## Generate ZIP deliverables

Run this from the repository root:

```bash
python3 examples/premium-website-showcase/scripts/package_demos.py
```

The script writes ZIP files to:

```text
examples/premium-website-showcase/demo-zips/
```

## Positioning

The intended message is simple: install the skills, provide a prompt/reference/spec, then use the workflow to generate local, inspectable frontend artifacts like these quickly.
