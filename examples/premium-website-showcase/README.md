# Premium Website Showcase

These demos are proof-of-output examples for the Codex Premium Website Skills library. They show the kind of polished local websites, hero sections, landing pages, and portfolios the workflow is meant to help produce quickly from prompts, references, specs, and iterative QA.

## Local preview

Open the gallery after cloning the repository:

```text
examples/premium-website-showcase/gallery/index.html
```

Open any demo directly, for example:

```text
examples/premium-website-showcase/demos/michael-smith-portfolio/preview.html
```

## Generate ZIP deliverables

```bash
python3 examples/premium-website-showcase/scripts/package_demos.py
```

ZIP files are written to `examples/premium-website-showcase/demo-zips/`.
